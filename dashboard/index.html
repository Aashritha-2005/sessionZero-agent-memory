About SessionZero

SessionZero is a trust-weighted memory layer for AI coding agents. It remembers past engineering decisions, checks whether they are still reliable, and only passes useful memory to the agent with a confidence label.

Ingest

Real decisions, commits, bug fixes, and architecture notes become memory units.

2

Store

Memories are written into Cognee Cloud's knowledge graph.

3

Recall

When an agent asks a question, SessionZero retrieves relevant memories.

4

Score trust

Each memory is scored using path, similarity, recency, and contradiction signals.

5

Decide

High-confidence memory is trusted. Low-confidence or contradicted memory is flagged.

6

Forget

Stale or superseded memory can be removed from the graph.

Why it matters

  • AI agents can remember old decisions.
  • But old decisions may become wrong.
  • SessionZero prevents stale memory from being trusted blindly.

Trust signals

Path
How connected the memory is in the graph.
Similarity
How relevant it is to the query.
Recency
How fresh the source decision is.
Contradiction
Whether a newer memory supersedes it.
